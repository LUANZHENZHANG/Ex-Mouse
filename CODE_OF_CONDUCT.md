# 社区行为准则

顺鼠欢迎不同经验、背景和观点的参与者。参与 Issue、Pull Request、讨论或其他项目空间时，
请遵守以下规则。

## 应有行为

- 围绕技术事实和可复现结果展开讨论。
- 尊重不同经验水平，给出具体、可执行的反馈。
- 接受合理的代码审查，并清楚说明自己的判断依据。
- 发现错误时及时更正，发生分歧时聚焦项目目标。
- 保护他人的隐私，不公开敏感信息。

## 不可接受的行为

- 人身攻击、侮辱、骚扰、歧视或威胁。
- 持续偏离主题、恶意抬杠或破坏正常协作。
- 未经允许公开他人的私人信息。
- 发布恶意代码、隐藏遥测或利用项目权限伤害用户。
- 其他在专业开源协作环境中明显不适当的行为。

## 执行

维护者可以编辑、隐藏或删除违反准则的内容，并视情况警告、临时限制或永久禁止参与者。
如需私下报告行为问题，请通过维护者的
[GitHub 主页](https://github.com/LUANZHENZHANG) 联系。维护者会尽量保护报告人的隐私。

本准则适用于项目仓库以及代表项目参与的其他公共空间。
