# 更新记录

本项目使用 `主版本.次版本` 形式记录当前发布版本。尚未发布的修改集中记录在
“未发布”一节。

## 未发布

- 暂无。

## 1.12 - 2026-06-11

### 新增

- 使用新的顺鼠 Ex-Mouse 图标。

### 调整

- 授权流程由辅助功能、输入监控和自动化三项缩减为只需辅助功能一项。
- 桌面切换改为直接发送系统快捷键，不再依赖 System Events。
- 合并滚动与触控板识别事件监听，保留原有滚动设备判断逻辑。

## 1.11 - 2026-06-10

### 新增

- 鼠标滚轮与触控板使用相反的自然滚动方向。
- 中键左右滑动切换桌面。
- 中键纵向滑动打开调度中心。
- 鼠标侧键切换上一个或下一个桌面。
- 菜单栏状态、设置和调试入口。
- 构建、安装、开机启动和卸载脚本。
- 可拖入 Applications 安装的 Apple Silicon DMG。
- 开源许可证、贡献指南、安全政策和社区协作模板。

### 调整

- 使用 `NextScrollBackend` 区分鼠标滚轮和触控板事件。
- 移除旧滚动后端、HID 探针和右键手势运行路径。
